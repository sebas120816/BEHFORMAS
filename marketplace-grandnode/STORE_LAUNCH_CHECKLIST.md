# Checklist de salida a producción BEH

## Funcionalidad

- [ ] Inicio, categorías, búsqueda, fichas, carrito y cuenta responden sin errores.
- [ ] Todos los productos tienen nombre, SKU, precio COP, descripción, medidas, material, garantía y fotografías.
- [ ] Los productos personalizados indican tiempos, anticipo y condiciones de retracto.
- [ ] Se probó un pedido completo desde móvil y escritorio.

## Configuración comercial

- [ ] Moneda principal COP y precios revisados.
- [ ] IVA y condiciones tributarias validados por contabilidad.
- [ ] Método de pago productivo configurado y probado.
- [ ] `Payments.CashOnDelivery` instalado y renombrado como transferencia o anticipo coordinado.
- [ ] El pedido de prueba queda en estado de pago pendiente, nunca pagado automáticamente.
- [ ] Los datos bancarios se comparten por un canal oficial y no están escritos en el código.
- [ ] Envíos, instalación, cobertura y recogida configurados.
- [ ] Correo transaccional configurado para pedidos y recuperación de contraseña.
- [ ] Prueba de SMTP enviada desde **Configuración > Cuentas de correo**.
- [ ] Datos de empresa, WhatsApp, dirección y horario actualizados.

## Operación

- [ ] Responsables definidos para productos, pedidos, clientes y contenido.
- [ ] Política de cambios, garantía, privacidad y términos publicados.
- [ ] Copia de seguridad automática verificada.
- [ ] Se realizó una restauración de prueba.
- [ ] Certificado HTTPS, dominio y vigilancia de proceso verificados.

## Antes de campañas

- [ ] Inventario y tiempos confirmados.
- [ ] Página de destino y productos anunciados revisados.
- [ ] Analítica y conversiones configuradas con consentimiento.
- [ ] Equipo comercial preparado para responder cotizaciones y WhatsApp.
